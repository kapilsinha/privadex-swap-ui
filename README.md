# PrivaDEX Swap UI

## Installation and run UI

- [Node.js](https://nodejs.org/en/)
- [Yarn](https://classic.yarnpkg.com/) or [npm](https://www.npmjs.com/)
- [MetaMask Browser Extension](https://metamask.io/download.html)

```bash
$ cd project

# Install the dependencies
$ yarn

# Run the web app
$ yarn start
```
